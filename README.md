# ARKit-Live2D

2020.10.23 尝试，成功，可实现类似元音的效果，使用的是live2d，素材是免费的

原Demo：https://github.com/mzyy94/ARKit-Live2D 

直接运行即可，由于文件较大，下载较慢

## Requirements
- iPhone and iPad models that support Face ID
- iOS 13.2 or higher version
- Xcode 11.3
